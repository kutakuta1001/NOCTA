## 0. CEOコンセプト（原文の要点）

- 「この色に浸る」ボタンを押すと、**簡単に書いて遊べる画面**が出てくる
- 書く土台（紙）や周りの風景は**その色のカラーパレットに合わせる**
- **幾何学模様や日本の伝統的な模様**をデフォルトで表示（参考画像: イスラムタイル文様の額×生成りの紙×青インクのカリグラフィ）
- ペンに特徴を出す: **カリグラフィ＋インクを使っているような描写・形**
- サムネイル紹介文と**HiNaコンセプトそのものにも反映**

## 1. 体験設計 —「筆あそび」

浸るモードを「純色を眺める」から「**その色の小さな書斎に入る**」へ進化させる。

画面構成（参考画像の構造を日本の意匠に翻訳）:
- **中央: 和紙** — 生成り（#F5EFE0）にその色を6%混ぜた紙。既存の粒子ノイズ（.grain）が和紙の肌理になる
- **周囲: かさね文様の額** — その色の「かさね2色＋本色」で描いた日本伝統文様のSVG帯。横長画面では左右、縦長画面では上下に配置（参考画像と同じ額装構造）
- **筆記: インクの物理** — 速度で太さが変わり、止めるとにじみ、書き続けるとかすれ、払いで先細る
- **銀猫との接続**: 猫が絵の具を踏んで足跡を残すように、ユーザーもその色のインクで跡を残す。「遊ぶことで人生を彩ろう」の実装

### 伝統文様（季節マッピング・SVGで動的生成）
データ追加不要。色の `season` から自動選択:
| 季節 | 文様 | SVG実装 |
|---|---|---|
| 春 | 麻の葉（あさのは） | 六角形＋放射線のline pattern |
| 夏 | 青海波（せいがいは） | 同心円弧の重なり |
| 秋 | 矢絣（やがすり） | 矢羽根のpolygon縦連 |
| 冬 | 市松（いちまつ） | 2色チェッカー |

配色 = `[本色, かさね1, かさね2]`（既存の kasane データを額に転用）＋ 地は紙色。線・塗りのopacityは0.5〜0.8で和紙に沈ませる。

### ペン（2種＋インク色4種）
- **筆（ふで）**: 丸ニブ。柔らかいradial gradientスタンプ（中心α0.85→縁α0.55）を40%オーバーラップで連打
- **カリグラフィ**: 斜め45°固定の楕円ニブ（rx=w, ry=w*0.35）。参考画像の書体の再現
- インク色: **本色 / かさね1 / かさね2 / 墨（#1C1A17）**。紙とのコントラスト比が1.8未満の淡色（桜色等）は自動で濃色版（color 60% + 墨 40% 混合・名前は同じ）に置換

### インク物理（Canvas 2D・実装パラメータ）
- ストローク: pointer eventsで点列 {x, y, t} を収集。線幅 `w = clamp(wMax − k·v, wMin, wMax)`（v=速度、EMAで平滑化）。wMax=14, wMin=2.5, k=0.035（px/ms比）目安
- **にじみ（インク溜まり）**: pointerdown時、および速度が閾値未満で120ms以上とどまった時に、半径 w*1.6 の淡いブロブ（α0.22のradial gradient）を追加
- **かすれ（ドライブラシ）**: インク残量 reservoir=1.0 がストローク距離で減衰（1200pxで0.35まで）。α×=reservoir、0.5未満でスタンプをランダム間引き（確率 = 0.5−reservoir）。新ストロークで全回復
- **払い**: pointerup時に終端速度が高ければ、進行方向へ1〜2スタンプ外挿しながら幅を急減
- 合成: `globalCompositeOperation='multiply'` でインクが紙に染みる質感
- 解像度: devicePixelRatio対応（cap 2）。`touch-action:none`

### UI（最小・既存iro-chipトーン）
下部バー: 色スウォッチ4個（丸・選択中は輪郭強調）／筆種2チップ（筆・カリグラフィ）／「まっさらに」（クリア）／「写しを残す」（PNG保存 = 紙＋文様額＋描線をオフスクリーン合成し `hare-{slug}.png` でダウンロード。外部送信なし）
コーナー: 色名（縦書き小・既存 .iro-name-v の縮小版）＋ かな。閉じるは既存 index-close。

### アクセシビリティ・互換
- ダイアログ開閉・ESC・フォーカストラップ・背面inertは既存 zukan-core の registerDialog をそのまま使用（immerse を asobi 画面に置換。「色面クリックで閉じる」挙動は**廃止**し、閉じるボタン＋ESCのみに変更 — canvasへの描画クリックと衝突するため）
- `prefers-reduced-motion`: にじみの追加アニメなし（静的描画のみ・機能は維持）
- キャンバスに aria-label「{色名}の紙。ポインタで書けます」

## 4. HiNaコンセプトへの反映（DESIGN-NOTES §7 追記）

HiNaの体験原則を明文化し、「遊び」を第4原則として追加:
1. ひとつずつ、静かに眺める（1画面1点）
2. 今日のひとつが毎日変わる（今日の一石/一花/一色）
3. 気に入ったものに肉球の印をつける（お気に入り）
4. **眺めるだけで終わらず、遊んで跡を残す**（Hare: 筆あそび。将来 Gyu=光にかざす・Ichi=一輪を活ける等、各作品に「遊びのレイヤー」を1つずつ足していく構想）

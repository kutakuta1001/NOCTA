# 日本の伝統色図鑑 設計書 — website/iro/

作成日: 2026-07-04
ステータス: CEOフル委任（確認不要）による自律実装
前提: 宝石図鑑v2・花図鑑v1（本日公開）と同系列の第7のツール。CEOの動機は書籍「日本の美しい言葉と配色」— 美しい言葉（色名・由来）と配色への関心。

## 1. コンセプト

「色に浸る」図鑑。宝石・花と同じ1画面1点の瞑想的スクロールだが、本作は**画像が不要で、セクション背景そのものがその伝統色に染まる**没入型。
言葉の美しさ（色名・読み・由来の物語）と配色（伝統色の組み合わせ）を核に据える。

- 記憶に残る一点: **色名の縦書き大組み**（Shippori Mincho）。色面の上に和書の背表紙のように立つ
- NOCTA継承: nav/footer/カバーは warm-black #0A0906 フレーム・粒子ノイズ・銀猫（フッターは silver-paint.png = 絵の具の猫）
- 色面上のテキストは背景輝度で墨色/生成り色に自動切替（コントラスト担保）

## 2. スコープ

### 収録: 伝統色48色（季節×12色）
春・夏・秋・冬の各12色。分類軸は「季節」と「色系統」（赤/桃/橙/茶/黄/緑/青/紫/白/鼠/黒）。
HEX値は一般に流通する慣用値を採用し、フッターに「出典により値は揺れる」旨を注記（リスト丸写しはせず、選定・物語・配色は自作）。

### 機能
1. **今日の一色**: 日付シード選出（gems/floraと同ロジック）。カバーに色見本＋色名＋読み
2. **目次オーバーレイ**: 季節チップ＋色系統チップ＋お気に入りフィルタ（gems/floraと同パターン・aria対応）
3. **HEX/RGBコピー**: チップをタップでクリップボードへ。「写しました」フィードバック
4. **配色（かさね）**: 各色に相性のよい伝統色2色を添える。タップでその色へジャンプ
5. **私のパレット**: お気に入り（肉球・localStorage）した色を目次上部に帯状パレットとして表示。タップでHEXコピー
6. **全画面鑑賞モード**: 色面タップで純色フルスクリーン＋縦書き色名。ESC/タップで復帰
7. キーボードナビ（↑↓/j/k）・サイドレール・JA/EN切替・reduced-motion対応

### 見送り（YAGNI）
CMYK表示・パレットの画像エクスポート・配色ジェネレーター・グラデーション生成・外部フォント追加

## 3. データ設計（iro-data.js）

```js
const NOCTA_IRO = [{
  name: "茜色",              // 色名（漢字）
  kana: "あかねいろ",
  romaji: "Akane-iro",
  hex: "#B7282E",           // 慣用値
  season: "秋",             // 春/夏/秋/冬
  family: "red",            // red/pink/orange/brown/yellow/green/blue/purple/white/gray/black
  kasane: ["黄金色", "朽葉色"],  // 配色（収録48色内の名前参照・レンダラーがhexを解決）
  storyJa: "茜の根で染めた、夕焼けの記憶を宿す赤。...",
  storyEn: "..."
}, ...]
```
- kasaneの参照整合性はテストで全件検証（存在しない色名参照をエラーにする）
- テキストのコントラストはWCAG相対輝度で判定（L>0.45→墨 #2A2118、以下→生成り #F2EBDD）

## 4. 実装方針

- website/iro/index.html + iro-data.js の自己完結2ファイル（still/gems/flora前例踏襲・共有JSなし）
- gems/floraのエンジン（i18n・目次・フィルタ・お気に入り・キーボード・レール・フォーカストラップ）を色向けに移植
- クリップボード: navigator.clipboard.writeText + 失敗時フォールバック（textarea/execCommand）
- apps-data.js 先頭に「伝統色図鑑」（NOCTA Tools — 07）を追加

## 5. テスト・検証

1. node --check / インラインJS構文検証
2. データ整合性: 必須フィールド・HEX形式・kasane参照解決・季節/系統の分布・名前重複
3. コントラスト: 全48色で自動判定した文字色とのコントラスト比が4.5:1以上であることを機械検証
4. Playwright実機: セクション数48・今日の一色・目次/フィルタ・コピー動作・パレット・全画面モード・i18n・コンソールエラー0
5. Codexレビュー2周 → website/iro/ と apps-data.js のみ commit・push

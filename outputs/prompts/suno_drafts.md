## 案A: 「森林のテクスチャーを音響設計に埋め込む」

### Style of Music（Sunoのstyle欄にコピペ）
```
J-pop, anime opening, 148bpm, A major, female vocal, piano, synth lead, strings, tight drums, nature ambience, uplifting, adventurous, bright
```

### 日本語ムード補足
イントロに森の環境音（鳥・葉・風）を薄くレイヤーし、そこから疾走感あるJポップに突入する構成。
電子音と自然音の対比が世界観の独自性を作る。
fhana・supercellにはない「空間的サウンドスケープ」が特徴。

### 仮歌詞（Sunoのlyrics欄にコピペ）

```
(forest ambience, birds, wind)

[Verse]
朝靄の中 踏み出した一歩
まだ見ぬ道が 輝いて見えた
葉の隙間から 光がこぼれて
胸の奥で 何かが動き出す

[Chorus]
ここから始まる 新しい物語
怖くないよ だって今 前を向いている
木々の声が 背中を押してくれる
NuWord 踏み出せ 世界はもう待ってる
```

## 案C: 「Bメロ転調で期待感を意図的に高める」

### Style of Music（Sunoのstyle欄にコピペ）
```
J-pop, anime opening, 148bpm, A major, female vocal, synth, piano, tight drums, 16th note synth sequence, key change bridge, uplifting, energetic, hopeful
```

### 日本語ムード補足
Aメロ（Aメジャー）→ Bメロ（F#マイナーに転調、緊張感・不安と期待の交差）→ サビ（Aメジャーに戻し解放）の感情arc。
fhanaが多用する転調テクニックをベースに、「決意の瞬間」をサビで表現する。
疾走感はリズムパターンとシンセシーケンスで作り、ストリングスは後半のみ投入。

### 仮歌詞（Sunoのlyrics欄にコピペ）
```
[Verse]
朝の空気が 冷たく澄んでいた
靴紐を結ぶ 指が少し震えて
でも顔を上げれば 森が広がっていて
ここから先が 私の物語

[Pre-Chorus]
(key change to F# minor)
本当にこれでいいのか 声が揺れる
でもこの高鳴りは 嘘じゃない
不安と期待が 溶け合う場所で
私は今 決める

[Chorus]
(back to A major)
走れ NuWord まだ見ぬ場所へ
答えは道の先にしかないから
恐れるな 全部を抱えたまま
世界へ飛び込め 今ここから
```

## Sunoでの生成手順

1. Suno を開き「Create」→「Custom Mode」をオン
2. 上記の Style of Music を「Style of Music」欄に貼り付け
3. 仮歌詞を「Lyrics」欄に貼り付け
4. 各案で2〜3パターン生成して聴き比べる
5. 方向性が決まったら案番号を教えてください

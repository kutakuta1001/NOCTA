07 — Gold Accent Usage Guide

✦ PRIMARY USE
  • • Section タグ（"About" "Services" 等）
  • • CTA ボタン（Contact / Works を見る）
  • • ニュース日付表示
  • • Service 02・04 の番号ラベル
✦ ACCENT USE
  • • About ビジュアルのノードドット
  • • 名刺コーナーライン
  • • フッターコピーライト文字
  • • Submitボタンのグラデーション起点
✦ AVOID
  • • 本文テキスト全体への使用
  • • Neon Lime と隣接・混在
  • • 4色以上同時使用のグラデーション
  • • ネオンライムより目立つ面積での使用

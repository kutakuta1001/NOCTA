02 — Color System

03 — Typography

サイズとコントラストで語る

強調はウェイトではなくサイズとシルバーのコントラストに委ねる。セリフ見出しは 600 が上限 ── 太字で押し切らない。本文は Noto Sans JP 300/400 を 14px で通す。

07 — Usage

守ること・避けること

Do

  • シルバー #B8B4AE を唯一のアクセントとして、少量に使う
  • 強調はウェイトより先にサイズとコントラストで
  • warm-black の余白とグレインノイズを残す（ブランド署名）
  • セクションタグ + シルバーの罫線プレフィックスで見出しを開く
  • プライマリ CTA はシルバーのピル、1ビューに最大1つ

Don't

  • 第二のアクセント色を足さない（gold / neon / lavender は廃止）
  • セリフ見出しを 600 より太くしない
  • 白背景・明るいグラデーション・オーロラを使わない
  • Space Grotesk / Inter を使わない（EB Garamond / Syne へ）
  • 多色グラデーションのロゴを復活させない

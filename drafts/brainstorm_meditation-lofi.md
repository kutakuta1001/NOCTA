作成日: 2026-04-26
フェーズ: 新プロジェクト企画（NuWordと並行）

---

## コンセプト

**テーマ**: 朝の瞑想・自己内省
**用途**: NOCTAツール（Seed / Verse / inner canvas）のアプリ内BGM
**方針**: ボーカルなし・純粋な環境音＋ピアノ。邪魔しない、でも空気を作る音。

**世界観キーワード**: 茶室・静寂・間（ま）・朝の光・湿った空気・呼吸

---

## 技術仕様

| 項目 | 値 |
|------|-----|
| ビットレート | 64〜96 kbps MP3 |
| 1ファイル上限 | 3MB以下 |
| 尺 | 2〜4分（ループ対応） |
| ボーカル | なし |
| 生成ツール | Suno v5（プロンプト生成 → CEO手動実行） |
| 用途 | NOCTAアプリ内BGM（ループ再生） |

**計算確認**: 96kbps × 4分 ≈ 2.75MB → 3MB以内OK

## Sunoプロンプト方針

- スタイルタグ: `lo-fi, ambient, meditation, Japanese aesthetic, instrumental`
- 楽器: `piano, nature sounds, atmospheric, binaural tones`
- 禁止: `vocals, lyrics, drums, upbeat, energetic`
- BPM目安: 50〜70（極めてゆっくり）
- 各トラックのプロンプトは `/phase1-suno` を改変するか個別に生成する

---

## アプリへの組み込み方針（要別途設計）

- ファイル置き場: `website/audio/meditation/` または IPFS
- 再生方式: 自動再生（ページ表示時）またはユーザー起動ボタン
- ループ: seamless loop（Web Audio API or `<audio loop>`）
- アプリごとの割当（案）:
  - Seed → 森の夜明け（創造・始まり）
  - Verse → 茶室の朝（言葉・静寂）
  - inner canvas → 内省（内向き・深層）

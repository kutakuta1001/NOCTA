## コード進行

### Aメロ（16小節 = 4パターン × 4回）
```
C# - G# - A#m - F#
（I  - V  - vi  - IV）
各コード2小節
```

### Bメロ（8小節 = 2パターン × 4回）
```
F# - G# - C# - A#m
（IV - V  - I  - vi）
各コード1小節 → 緊張感を高める短いサイクル
```

### サビ（16小節 = 4パターン × 4回）
```
C# - A#m - F# - G#
（I  - vi  - IV - V）
各コード2小節
```

### 落ちサビ（8小節 = 2パターン × 4回）
```
A#m - F# - C# - G#
（vi  - IV - I  - V）
各コード1小節 → マイナー感から始まり解決に向かう
```

### ラスサビ（サビと同進行 + 最終2小節のみ変更）
```
C# - A#m - F# - G#  × 3
C# - F#  - G# - C#  （最終2小節: I - IV - V - I で完全解決）
```

## トラック構成（Studio One トラック一覧）

### 1. Drums
- **Aメロ/Bメロ**: タイトな8ビート。キック: 1・3拍、スネア: 2・4拍
- **サビ/ラスサビ**: 16分シンコペーション。キック: 1・2拍目裏・3拍・4拍目裏
- **落ちサビ**: ミュート（ドラムなし）
- **推奨音源**: Studio One付属 Impact XT / BFD3

### 2. Bass
- **Aメロ**: ルート音を4分音符で刻む（C#2 → G#1 → A#1 → F#1）
- **Bメロ**: 8分音符でベースランを追加（F#1-G#1-A#1-B#1 の上行ライン）
- **サビ**: ルート＋5度のパワーベース（C#2-G#2のオクターブ跳躍）
- **落ちサビ**: 全休符
- **ラスサビ**: サビと同じ + 最後4小節でスラップ風に強調

### 3. Piano（Lead）
- **イントロ**: C#メジャースケールのアルペジオ（C#4-E#4-G#4-C#5 上行）
- **Aメロ**: コードを8分音符カッティング
- **落ちサビ**: 主旋律を弾く（ボーカルと対話）。音域: C#4〜G#5
- **アウトロ**: イントロのアルペジオに戻る

### 4. Synth Lead
- **サビのみ使用**: ソーリード（Saw波 + フィルター）でメロディをダブリング
- カットオフ: 4kHz、レゾナンス: 20%、アタック: 8ms
- **ラスサビ**: オクターブ上でユニゾン

### 5. Synth Pad
- **Bメロ以降**: ウォームなパッド（C#メジャー和音を長音で保持）
- **アタック: 400ms、リリース: 800ms**（滑らかに出入り）

### 6. Strings
- **サビ後半8小節から薄く投入**（最初は pp）
- **ラスサビ**: ff で全開。ダウンビート強調のマルカート奏法
- **音域**: ヴァイオリン: G#4〜C#6 / チェロ: C#2〜G#3

### 7. 森林SE（Ambient Layer）
- **イントロ**: 鳥のさえずり + 葉ずれの風 + 小川の水音を4小節でフェードイン
- **落ちサビ頭**: 同SEを -18dB で薄くかぶせる（0.5小節でフェードアウト）
- **アウトロ**: 再度フェードイン、最後は森林SEのみでフェードアウト
- **用意方法**: Freesound.orgで "forest ambience" で取得 / Studio One付属SE素材

## 1. 直近3ヶ月のチャート傾向（2025年12月〜2026年3月）

### BPM帯
- **主流帯: 138〜158 BPM**
  - アニメOPタイアップ曲はこの帯域に集中（疾走感・爽快感の需要が高い）
  - 特に144〜152 BPMが「速すぎず・遅すぎず」として多用されている
- サブ帯: 96〜112 BPM（Bメロで緩急をつける構成に使用）

### キー
- **メジャー優勢期**: A♭メジャー・Eメジャー・Dメジャーが多い
  - 女性ボーカルの高音域（hiA〜hiE）が映えるキー選択が目立つ
  - Maiの音域（C4〜C6）を活かすなら **Aメジャー または Bメジャー** が最適
- マイナー曲も一定数あるが、「前向き・疾走」テーマではメジャーが圧倒的

### 楽曲構成の特徴
- **標準構成**: イントロ(4小節) → Aメロ(16) → Bメロ(8) → サビ(16) → Aメロ(16) → Bメロ(8) → サビ(16) → Cメロ/大サビ(8〜16) → ラスサビ(16) → アウトロ(4)
- **トレンドポイント**:
  - イントロ短縮傾向（4小節以内でサビ的フックを出す曲が増加）
  - Cメロで転調（半音〜全音上）してラスサビで感情的クライマックスを作る構成が多い
  - サビ終わりに「落ちサビ」（ピアノ/アコギ一本に絞る）を挟む曲が支持される
- **アレンジ面**: ストリングスとシンセの混在が標準。ドラムはタイトでキックとスネアのコンプが強め

## 3. 今避けるべきクリシェ3パターン

### クリシェ①: 「過剰なストリングス起爆型サビ」
- 2023〜2024年に多発した「サビでストリングスが爆発するだけ」の構成
- 現在は飽和しており、逆にサビで引き算するアレンジが新鮮に聴こえる
- **NuWordへの適用**: サビの第一フレーズはシンプルに。ストリングスは後半サビで段階投入する

### クリシェ②: 「4つ打ちキック+スネア2・4拍のみ」の単調ビート
- EDM由来の4つ打ちパターンはアニメ系では陳腐化が進んでいる
- タイトなハーフタイムシャッフルやシンコペーション入りパターンが差別化につながる
- **NuWordへの適用**: Aメロはタイトな8ビート、サビで16分シンコペーション入りパターンに切り替える

### クリシェ③: 「君のために/出会えたこと/永遠に続く」系の歌詞ワード
- 「永遠」「運命」は既にCEOが禁止ワード指定済みで適切な判断
- さらに「輝く未来」「翼を広げて」も現在は陳腐化のリスクあり
- **NuWordへの適用**: 「森林から冒険が始まる」という具体的な情景描写で抽象ワードを回避する

## 4. 差別化の方向性提案

### 案A: 「森林のテクスチャーを音響設計に埋め込む」
- 木漏れ日・葉の揺れ・鳥のさえずりを加工したサウンドレイヤーをイントロとCメロに忍ばせる
- 電子音主体のアレンジにナチュラル系SE（アンビエント）を薄くかぶせることで世界観の独自性を確立
- BPM: 148、キー: Aメジャー、構成: イントロ4小節（自然音SE）→ 通常のJポップ構成
- **強み**: 参考3アーティストにない視覚的・空間的サウンドスケープ

### 案B: 「落ちサビで世界観の核心を語る」
- 通常の盛り上がり曲と逆行して、最も重要な歌詞を落ちサビ（ピアノ1本+ボーカル）に置く
- ラスサビで全楽器投入してカタルシスを作る「静→動」の感情設計
- BPM: 144、キー: Eメジャー、落ちサビ: Cマイナー（短調転調で緊張感）→ ラスサビ: Eメジャー復帰
- **強み**: ろんのスタイルと親和性が高く、ターゲット（始まりを体感したいアーティスト）の心理に刺さる

### 案C: 「Bメロ転調で期待感を意図的に高める」
- Aメロ: Aメジャー → Bメロ: F#マイナー（平行短調）→ サビ: Aメジャーに戻す
- Bメロで短調に落として「まだ答えが見えない」緊張感を作り、サビで解放する設計
- 歌詞は「不安と期待が交差する」心理状態をBメロで表現し、サビで決意に転換
- **強み**: fhanaが多用する転調テクニックを参考にしつつ、NuWood独自の感情arc設計

## 総括メモ（music-spec-writer への引き継ぎ用）

- 推奨BPM: 144〜152
- 推奨キー: Aメジャー または Eメジャー
- 構成の核心: 落ちサビ or Bメロ転調による緊急感の演出
- Maiの音域を活かすキー設定が重要（C4〜C6、hiA〜hiCが特に映える）
- 疾走感の実現方法: ストリングスではなくリズムパターンとシンセシーケンスで作る

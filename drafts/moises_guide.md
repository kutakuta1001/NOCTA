# Moises.aiでSuno音源を分離する手順

## 目的
SunoのMP3からドラム・ベース・コード・ボーカルを
パーツ分離してStudio Oneで使えるようにする。

## 手順

### STEP 1: アカウント作成
1. moises.ai をブラウザで開く
2. Google/Apple/メールでアカウント作成してログイン

### STEP 2: MP3をアップロード
1. ダッシュボードの「Upload Track」をクリック
2. SunoでダウンロードしたMP3を選択してアップロード
3. 無料プランは月あたりのアップロード数に上限あり

### STEP 3: 分離タイプを選ぶ
以下の4分離を選ぶ（コード・ベース・リズム解析が目的のため）:
- Vocal（ボーカル）
- Drums（ドラム）
- Bass（ベース）
- Other（コード・パッドなど残り全部）

「Separate」または「Start」を押して数十秒〜数分待つ。

### STEP 4: パートを確認する
ミキサー画面が開いたら各パートをソロ再生して確認:
- Bassだけ → ベースラインが聴こえるか
- Drumsだけ → リズムパターンが聴こえるか
- Otherだけ → コード感が聴こえるか

### STEP 5: ダウンロード
- 無料プラン → MP3形式でダウンロード
- 有料プラン → WAV形式でダウンロード（Studio One推奨）
- 「Download all stems」で全パート一括ダウンロード

### STEP 6: Studio Oneに読み込む
1. Studio Oneで新規ソングを作成
2. ダウンロードした以下のファイルをドラッグ＆ドロップ:
   - Bass.mp3（ベースの参考として）
   - Drums.mp3（リズムの参考として）
   - Other.mp3（コード感の参考として）
3. Vocal.mp3は参考程度に置く
   （メロディはオリジナルで作るため）

### STEP 7: Chord Trackでコード検出
1. Other.mp3（コード・パッド成分）をChord Trackにドロップ
2. 右クリック→「コードを検出」を実行
3. コード進行が表示されたらmusic_spec.mdと照合する

## Studio One での使い分け
| パート | 用途 |
|---|---|
| Drums.mp3 | リズムの参考・そのまま使ってもよい |
| Bass.mp3 | ベースラインの参考・そのまま使ってもよい |
| Other.mp3 | コード感の参考・Chord Track検出に使う |
| Vocal.mp3 | 参考程度（メロディはオリジナルで作る） |

## 注意事項
- 商用利用する場合はSunoのProプラン（月20ドル）が必要
- 無料プランはMP3のみ・月の利用上限あり
- 分離精度は曲によって異なる（ベースとキックが混ざる場合あり）

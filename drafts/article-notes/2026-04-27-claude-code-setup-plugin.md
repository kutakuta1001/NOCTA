## 要約（3〜5行）
Anthropic公式プラグイン「claude-code-setup」がリリースされた。
プロジェクトを自動スキャンして、最適なhooks・skills・MCPサーバー・SubAgent構成をシニアエンジニア並みの精度で提案してくれる。
設定手順をステップガイド形式で案内するため、初心者だけでなく既存設定の棚卸しにも有用。

## 主なポイント
- プロジェクトを自動スキャンして hooks / skills / MCP / subagents の最適構成を提案
- 設定手順をステップガイド形式で案内
- CLAUDE.md に既記載の skill-creator プラグインとは別物（設定全般の自動化に特化）

## NOCTAへの関連メモ
NOCTAの現在のCLAUDE.md・Skills・Hooks・MCP設定の最適性チェックに活用できる可能性がある。導入してNOCTAリポジトリをスキャンさせると新しい改善提案が得られそう。

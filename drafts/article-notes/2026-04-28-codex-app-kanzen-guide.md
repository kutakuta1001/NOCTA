## 要約（3〜5行）
OpenAI の Codex アプリ（ChatGPT Plus 加入者は追加費用なし）の解説スレッド。
GPT-5.5・GPT Image 2・Computer Use・Memory・90+プラグインを搭載したエージェント型アプリで、ChatGPT アプリとは別物。
筆者は Claude Code をメイン開発エージェントとして使いながら、OpenAI モデルが必要な場面の窓口として Codex アプリを固定ポジションにしている。

## 主なポイント
- ChatGPT Plus（月$20〜）に加入中なら追加費用なしで Codex アプリが使える
- GPT Image 2：Image Arena 1位・4K対応の最強画像生成AI（NOCTAのビジュアル生成に関連）
- Computer Use：PC 操作の自動化が可能
- Memory：プロジェクトスタイルガイドを学習させると毎回説明不要
- Claude Code がメイン・Codex が OpenAI モデルの窓口という二刀流運用が最強という結論

## NOCTAへの関連メモ
NOCTA は Claude Code メイン運用で一致。GPT Image 2 は R-04 の外部ビジュアル生成ツールとして検討余地あり（現在は Midjourney / Dreamina 中心）。Codex アプリの二刀流は NOCTA には直接不要だが、GPT Image 2 の品質向上は注目。

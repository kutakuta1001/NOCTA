## 要約（3〜5行）
非エンジニアが Claude Code で副業自動化システムを構築している実践ログを発信しているアカウント。
CLAUDE.md / Hooks / MCP / /clear /compact / Plan mode / Anthropic SDK / LangGraph / ablation実験 / production運用ログを継続的に発信している。
「AIで簡単に稼ぐ」ではなく、実際に動くシステムを丁寧に構築する姿勢が参考になる。

## 主なポイント
- 非エンジニアでも CLAUDE.md + Hooks + MCP で本格的な自動化が可能
- ablation実験（変数を一つずつ変えて効果を測る実証的アプローチ）を実践
- LangGraph との組み合わせも検討している
- production運用ログを公開しているため、実際の課題と解決策が参照できる

## NOCTAへの関連メモ
NOCTAも非エンジニアCEO主導の運用という意味で同様のアプローチ。フォローして運用ログを定期的に参照する価値あり。ablation実験の手法はNOCTAのAgent Teams設計改善にも応用できる。

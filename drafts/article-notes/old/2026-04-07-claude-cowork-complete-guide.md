## 要約（3〜5行）

Claude Cowork（Claude Desktop のエージェント機能）の全機能を約30,000文字で徹底解説したツイートスレッド。「AIに質問する」時代を終わらせ「AIが仕事を終わらせる」時代への移行を宣言。非エンジニアがターミナル不要で Claude Code 相当の自律タスク実行ができる点を強調。料金プランから具体的なプロンプト例まで網羅。

## 主なポイント

- **3つの使い方の違い**: チャット（相談相手）→ Cowork（同僚）→ Code（専属エンジニア）の役割を整理。Cowork は非エンジニア向けの Claude Code 相当
- **10の機能**: ファイル操作・Excel生成・PowerPoint作成・PDF入力・Web調査自動化・Gmail/カレンダー連携・マルチステップ並列実行・スケジュールタスク・スキル&プラグイン
- **コスト**: Pro（$20/月）から利用可。ただし Cowork はチャットの10〜30倍トークン消費。ヘビーユーザーは Max 5x（$100/月）推奨
- **並列性の本質**: 「AIは速さではなく並列性が価値。3つ同時に動かせば生産性3倍」
- **スキル&プラグイン**: 定型業務を Cowork に覚えさせて再利用・組織共有できる仕組み（Claude Code の SKILL.md に相当）

## NOCTAへの関連メモ

NOCTA は Claude Code ベースで同等機能を既に実現済み（スタッフゼロ型 AI 事業 OS）。Cowork は非エンジニア向けの Claude Code 代替として有用だが NOCTA には直接不要。ただし「新人マネジメント型プロンプト」（計画を先に見せてから実行）の発想は NOCTA の R-13「対話してから作業する」と一致しており、スキル設計の参考になる。

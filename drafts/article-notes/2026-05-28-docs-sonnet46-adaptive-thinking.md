## 要約（3〜5行）
公式モデル一覧ページで Claude Sonnet 4.6 が適応的思考（Adaptive Thinking）に対応していることが確認された。
NOCTA の R-09 では「Opus 4.7 が適応的思考あり」とのみ記載されており、Sonnet 4.6 の適応的思考対応が漏れていた。
モデル能力の正確な把握のために R-09 の説明更新が推奨される。

## 主なポイント
- Opus 4.7: 拡張思考(×) + 適応的思考(○)
- Sonnet 4.6: 拡張思考(○) + 適応的思考(○)  ← R-09 未記載
- Haiku 4.5: 拡張思考(○) + 適応的思考(×)
- 適応的思考（Adaptive Thinking）は状況に応じて思考深度を自動調整する機能

## NOCTAへの関連メモ
R-09 の Sonnet 4.6 説明を「拡張思考・適応的思考の両方に対応」と更新すると正確になる。CEOが判断して手動で更新する。

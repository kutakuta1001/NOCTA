## 要約（3〜5行）
Claude & Cursor用のMCPサーバーを横断的に検索できるオープンソースのディレクトリを公開。ローカルDB・Slack・その他ツールへの接続に特化したコミュニティ製MCPを一覧できる。ワークフロー最適化に使えるMCPサーバーを効率的に探せる。

## 主なポイント
- コミュニティ製MCPサーバーの横断検索ディレクトリ
- ローカルDB・Slack・生産性ツールなどへの接続MCPを収録
- Claude & Cursor の両方に対応
- GitHub で公開されているオープンソースプロジェクト

## NOCTAへの関連メモ
NOCTAではxmcp（X公式MCP）を使用中。このディレクトリで音楽制作関連MCP（Studio One連携・Synthesizer V連携・IPFSアップロード等）が見つかる可能性がある。新MCP追加時は R-03・ツール総数80以下のルールに従いdisabledMcpServersで管理する。

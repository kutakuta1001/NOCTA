## 要約（3〜5行）
「人間のための」デザインシステムだけでは不十分という前提で、AIコーディングエージェントが規約準拠のUIを生成・検証できる3層アーキテクチャ（DESIGN.md/CLAUDE.md → JSON契約仕様 → CI自動検証）を持つデザインシステム。MCPサーバー経由でトークン検索・ルール検証・コンポーネント取得ができる。TypeScript + Tailwind CSSベース、MITライセンス。

## 主なポイント
- Layer1: DESIGN.md（AI向けデザイン憲法）+ CLAUDE.md（作業手順書）
- Layer2: JSON契約仕様（99トークン・99ルール・40コンポーネント）
- Layer3: Playwright + axe-coreによるCI自動検証、99禁止パターンの自動検出
- `claude mcp add melta-ui -- npx -y melta-ds-mcp` でMCP統合可能

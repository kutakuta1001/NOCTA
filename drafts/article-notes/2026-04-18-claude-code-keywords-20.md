## 要約（3〜5行）
Claude Code を使いこなすための重要キーワード20選を網羅したツイート。CLAUDE.md・Skills・Sub Agents・settings.json・Rules・MCP・Hooks・Routines・remote control・CLI・Permission Modes・Memory など、Claude Code の全機能領域をカバーしている。入門者から上級者まで参照できる全体像のチートシートとして機能する。

## 主なポイント
- CLAUDE.md: エージェントへの常時有効ルール定義
- Skills: 状況限定ワークフローのスキルファイル（`~/.claude/commands/`）
- Sub Agents: 並列・委譲タスク実行
- Hooks: ツール呼び出し前後の自動処理
- Permission Modes: dontAsk / bypassPermissions / acceptEdits 等
- Memory: 会話をまたいだ記憶保持（auto memory システム）
- Routines: 定型的な繰り返しタスクの自動化

## NOCTAへの関連メモ
NOCTA の SLASH COMMANDS テーブルと対照すると、Routines と remote control の活用がまだ十分でない。特に remote control（Claude Code を外部から起動するパターン）はフェーズ自動移行や定時レポート生成への応用余地がある。

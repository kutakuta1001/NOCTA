## 要約

国内議事録ツール「Rimo」が CLI・MCP を公開し、Claude Code から議事録データを権限管理ごと参照できるようになった。国内初のAIエージェント連携議事録ツール。

## 主なポイント

- Rimo CLI・MCPサーバーが公開（OSS）
- Claude Code から議事録データへのアクセスが可能
- 権限管理ごと接続（ユーザーごとに参照範囲を制御）

## NOCTAへの関連メモ

NOCTAはチームミーティングがないため直接の用途はない。MCP を通じた外部データソース連携のアーキテクチャ例として参考になる（将来的に音楽制作の外部ツールと MCP 連携する場合の設計ヒント）。

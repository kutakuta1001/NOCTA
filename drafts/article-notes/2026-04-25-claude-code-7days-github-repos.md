## 要約（3〜5行）
Claude Codeを最短で学ぶにはチュートリアルより厳選されたGitHubリポジトリを読む方が効果的。筆頭は「Awesome Claude Code」でエージェント・コマンド・hooks・CLAUDE.md・完全なワークフローのインデックスとして機能する。（ポルトガル語ツイート）

## 主なポイント
- Awesome Claude Code: エージェント・コマンド・hooks・CLAUDE.md・フローの総合インデックス
- 実際のリポジトリから設計パターンを学ぶアプローチが効率的
- 7日間の独習ルートとして紹介

## NOCTAへの関連メモ
NOCTAのCLAUDE.md・スキル体系・Agent Teamsは「Awesome Claude Code」で紹介されているような先進的パターンをすでに実装済みと考えられる。新しいスキルパターンやhooks設計のアイデアを探すときに「Awesome Claude Code」リポジトリを参照すると効率的。

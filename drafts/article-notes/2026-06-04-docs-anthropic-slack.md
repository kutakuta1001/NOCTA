## 要約

Claude Code の Slack 連携が正式機能として公開。Slack で `@Claude` にメンションするとバグレポートをルーティングしてプルリクエストを自動作成するワークフローが実現できる。

## 主なポイント

- Slack ワークスペースに Claude Code を統合可能
- `@Claude` メンションでタスクをルーティング → 自動PR作成
- チャット → コードへのワークフローを非同期で実現

## NOCTAへの関連メモ

NOCTAは個人プロジェクトのため Slack は未使用。将来チームで制作する場合や、外部コラボレーターとの連携時に参照。現時点では不要。

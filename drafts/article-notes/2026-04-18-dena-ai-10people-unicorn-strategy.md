## 要約（3〜5行）
DeNA南場智子会長が「10人1組でユニコーン（時価総額1,000億円超）を量産する」戦略を宣言。Devin（自律型AIエンジニア）を全社2,000人超に導入し、開発の95%をAIが担い人間5%というモデルを実現。「並列化できる定型タスク → Devin 80%」→「仕上げ → Claude Code 15%」→「最終判断 → 人間5%」というハイブリッド運用が特徴。

## 主なポイント
- 「少人数×AI」で大企業と同等プロダクトを作るビジョン
- Devin（Cognition AI）: 自律型AIエンジニア。GitHubタスク→実装→PR提出を自動実行
- ハイブリッド運用: Devin 80%（定型・並列）→ Claude Code 15%（複雑・対話）→ 人間 5%
- 2027年度新卒採用に「AIジェネラリスト（年収600万〜）」コース新設
- 非エンジニアでもDevin×AIで8倍業務高速化を実現

## NOCTAへの関連メモ
NOCTAの「CEOとAIエージェントチームで完結」モデルと同じ思想。現在はClaude Codeだけだが、将来的に定型繰り返し作業（MIDI変換・フォーマット変換等）をDevin等の自律型エージェントに委ねるハイブリッド運用は検討価値がある。「AI×音楽制作」の少人数モデルとしての事業参考にもなる。

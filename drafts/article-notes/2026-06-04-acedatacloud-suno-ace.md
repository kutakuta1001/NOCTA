## 要約

「Ace に Suno の音楽生成がフル対応した。API 1回の呼び出し、またはスタジオで直接、完全なトラックが生成できる。」Ace Data Cloud のサービス紹介。

## 主なポイント

- Ace プラットフォームで Suno API 経由の完全トラック生成が可能
- API呼び出し1回またはスタジオ GUI から操作
- Suno のアクセス手段が増えた（API統合）

## NOCTAへの関連メモ

NOCTA は R-12 で Suno + LALAL.AI 分離活用を採用済み。Ace 経由の Suno API は価格・品質を確認してから代替として評価する。ACE-Step（OSSモデル）と混同しないよう注意（Ace DataCloud は別サービス）。

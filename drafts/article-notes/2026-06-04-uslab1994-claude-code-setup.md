## 要約

Anthropic公式の `claude-code-setup` プラグインを入れると、コードベースを読んで「このファイルを読ませろ」「このhooksを入れろ」「CLAUDE.mdにこれを書け」という構成を一括提案してくれる。

## 主なポイント

- プラグイン名: `claude-code-setup`（Anthropic公式）
- コードベースを解析して CLAUDE.md・hooks の初期構成を自動生成
- 新プロジェクト立ち上げ時の CLAUDE.md 作成コストを大幅削減

## NOCTAへの関連メモ

NOCTAは既に CLAUDE.md が充実しているため即時効果は薄いが、2曲目以降の新プロジェクトや NuWorld の CLAUDE.md 更新時に試す価値がある。Claude Code Max の Plugin Marketplace で確認する。
